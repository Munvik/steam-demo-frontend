# SteamDemoFrontend

I made this project for education of several technologies and frontend basics.

It is a simple Steam-like frontend where I practice:

## Why this project exists

This project was created to improve frontend skills in:

- building modern UI with React and Material UI
- creating reusable components and clean page structure
- using TypeScript for safer frontend code
- working with React Router and page navigation
- handling local state and UI flow simulations
- connecting frontend to backend APIs when needed

## Tech stack

- React 19
- TypeScript
- Vite
- React Router
- React Query
- Docker + Nginx

## Personal note

This is my second project in React.

I used a dose of AI to build the page faster, but a lot of parts are hand written or manually modified by me.

Main idea is to improve my frontend skills step by step, not to build a production app.
